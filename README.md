📊 ANOVA Analysis of Financial Statements using Python

As part of my financial analytics coursework, I performed Analysis of Variance (ANOVA) on financial data using Python.

The dataset was imported from a CSV file and analyzed using the statsmodels library.

🔹 Steps performed:

Imported financial dataset using pandas

Built an OLS regression model

Performed ANOVA using statsmodels.anova_lm()

📈 Key Insight:
The ANOVA results show a statistically significant relationship between Equity Capital and Total Assets.

🛠 Tools Used:

Python

Pandas

Statsmodels
